# Hospital Management System

A C# console application that manages hospital patients, wards, and beds.

## Features
- Add new patients to available beds
- Discharge patients
- Search for patients by name
- View ward occupancy
- Display all patients

## How to Run
1. Download the code
2. Open in Visual Studio
3. Press F5 to run
